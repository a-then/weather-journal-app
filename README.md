# weather-journal-app
An asynchronous web app that uses Web API and user data to dynamically update the UI for a Weather-Journal App.

### To run server you'll need:
  - node.js
  - express
  - cors

## Built with
  - JavaScript
  - Node.js
  - Express
  - Cors
